SELECTOR - RETURN TYPE

getElementById - single element
getElementByClassName - HTML Collection
getElementByTagName - HTML Collection
querySelector - first element of given type
querySelectorAll - Node List


Q. Differences between HTML Collection and Node List:
HTML Collection:
1. it is live (changes in elements after getting collection are reflected)
2. we can reference the elements by index, id, or by name

Node List:
1. it is static
2. reference the elements only by index
